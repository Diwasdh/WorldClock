## Prerequisites

- Node.js 16+
- Bun, npm, or yarn package manager

## Installation

```bash
# Clone the repository
git clone https://github.com/Diwasdh/WorldClock.git


# Navigate to project directory
cd world_clock

# Install dependencies (choose one)
bun install
# or
yarn install
# or
npm install
```

## Getting Started

```bash
# Run development server
bun run dev
# or
yarn dev
# or
npm run dev
```
